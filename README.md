# 📝 Todo List

A simple and responsive Todo List web application built using **HTML**, **CSS**, and **JavaScript**. Users can add, complete, and delete tasks with a clean and user-friendly interface.

## ✨ Features

- ➕ Add new tasks
- 📅 Set a due date for each task
- ✅ Mark tasks as completed using a checkbox
- 🗑️ Delete tasks
- 🎨 Responsive and modern UI

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
